# 🌸 时光的回响：记忆编织者 (Time's Echo: Memory Weaver)

> **2026 腾讯云黑客松（Tencent Cloud Hackathon）公益赛道作品**
> 
> *一款面向阿尔茨海默病预防与适老化关怀的唯美治愈系互动拼图网页游戏。*
> *通过温和的回忆疗法（Reminiscence Therapy）唤醒昔日温暖，用爱与陪伴包容他们的“遗忘”。*

---

## 💡 项目设计理念

在中国，有超过 **1000 万** 老年人正承受着阿尔茨海默病（老年痴呆症）的困扰。认知训练、生活干预、耐心的陪伴以及**回忆疗法（Reminiscence Therapy）**能有效延缓认知衰退。

《时光的回响：记忆编织者》以此为切入点，将阿尔茨海默病患者常见的“认知迷雾”、“记忆失焦”、“思绪断裂”及“精神孤独”等症状，转化为游戏中的**首部曲重构机制**，并以老式收音机、复古照相机、足踏缝纫机、煤油马灯这四个满载时代记忆的旧物为载体，带给玩家深度的共情与人文关怀。

---

## 🛠️ 核心交互系统：两部曲重构

游戏为四个纪念物件定制了截然不同的首部曲交互，生动对应阿兹海默症的四种临床干预手段：

### 📻 物件一：老式收音机 ——【拨开认知迷雾】
* **首部曲（拂尘）**：像轻抚长辈额头那般，在左侧物件上反复揉搓扫拭，拂去重重叠叠的迷雾（认知除尘）。
* **二部曲（拼图）**：将木质外壳、喇叭网罩、表盘与黄铜旋钮归位。
* **情感互动玩具**：左右滑动调频滑块，将频率拨动到 **98.7MHz** 黄金记忆频段，即可滤去电磁杂音，流淌出温暖清澈的八音盒民谣《送别》。

### 📸 物件二：复古照相机 ——【虚像镜头调焦】
* **首部曲（调焦）**：拖拽调焦环，让左侧失焦的复古镜头与齿轮从高斯模糊中渐渐清晰，拉近失落记忆的焦距。
* **二部曲（拼图）**：装配相机机身、光学镜头、快门旋钮与皮质肩带。
* **情感互动玩具**：点击快门，伴随清脆的机械快门与感光片弹跳声，机身触发高亮闪光，冲印出一张记录着“1985年春天野餐全家福”的拍立得手绘老照片。

### 🧵 物件三：足踏缝纫机 ——【缝合记忆引线】
* **首部曲（穿线）**：顺次点击金色闪烁的 `1➔2➔3➔4` 定位穿针孔，拉起明红色的心愿缝合线，连结脑海散落的线索。
* **二部曲（拼图）**：装配铸铁主臂、惯性皮带轮、压脚针头与红色木质线轴。
* **情感互动玩具**：长按足踏板运转缝纫机，触发“哒哒哒哒”极具机械打击感的连续踏板声效，针脚计数满 100 针后即可织成一件温暖的大红棉袄。

### 🕯️ 物件四：煤油马灯 ——【擦亮心中微光】
* **首部曲（燃灯）**：在右侧擦火带上向右快速划动火柴，重燃心中的温热微光，驱散冰冷孤独的黑暗脑海。
* **二部曲（拼图）**：装配铁铸底座、防风玻璃罩、防护网罩、调节阀顶盖与铁挽手。
* **情感互动玩具**：左右拖动阀门滑块改变亮度，当亮度提升至 80% 以上时，玻璃灯罩瞬间通透，马灯长明，温热全局。

## 🌟 核心技术亮点

### 1. 🎵 物理合成音效引擎 (Zero-Asset Web Audio API)
为了追求极速加载和无网络卡顿的极致体验，项目**未使用任何外部 MP3/WAV 资源**，所有音频皆由 JavaScript 动态实时合成：
* **Lo-Fi 氛围背景乐**：通过 `sound.js` 混合 Triangle 与 Sine 波形，动态合成温润的 Rhodes 钢琴 **Major 9th (大九) 爵士和弦序列**，并利用低频振荡器 (LFO) 调制音轨频率，完美复刻模拟磁带的 Wow & Flutter (物理抖晃) 慵懒颤音。
* **黑胶唱片刮擦噪声**：使用 Pink Noise (粉红噪声) 滤波器阻尼白噪声，并随机注入瞬时脉冲（Impulse Pops），营造深夜围炉长谈的旧物复古质感。
* **物理级打击反馈声**：快门弹簧声、缝纫机踩踏走线声、风铃咬合扣锁音等，均使用高精度包络发生器（Envelope Generator）实时算力渲染。
* **自愈机制**：内置 Autoplay-Blocked (自动播放挂起) 智能捕获，只要用户在屏幕上进行首次触摸、点击或键盘交互，即可秒级无缝重启音频上下文，确保静音拦截自愈。

### 2. 🗣️ 适老化无障碍配音 (SpeechSynthesis)
* 引入 Web Speech 朗读功能，专门过滤掉了修饰性的 Emoji 字符，提取纯净的自传文本。
* 适配了高兼容性的 Xiaoxiao / Huihui 柔和磁性女声声线，调低音调 (`0.95`)，放缓语速 (`0.82`)，模拟祖母叙述时的温暖沙哑 cadence。

### 3. 🛡️ 严格的安全性与环境兼容性 (Trusted Types)
* 针对现代 Chromium 浏览器严格的安全规范，在 `game.js` 顶部引入了 resilient Named Trusted Types 策略，防范任何 HTML 注入（DOM XSS）安全限制，无报错、零阻断。

### 4. 💎 极致美学的 Glassmorphic 界面设计
* 采用 HSL 调和色彩体系，深沉静谧的黑夜阁楼底色点缀大片柔粉（`#d48a85`）与金砂（`#e5a93b`）辉光。
* 配合毛玻璃卡片（`backdrop-filter: blur`）及粒子缓动漂浮引擎，使整个游戏充满温润、包容的情感触觉。

## 🚀 腾讯云 Cloud Studio 部署与预览指南

本项目为纯前端静态 Web 应用，非常适合在腾讯云 **Cloud Studio** 开发环境中一键托管、协作和运行：

### 1. 导入项目
将本仓库导入至您的 **Tencent Cloud Studio** 空间中。

### 2. 极速依赖安装
在 Cloud Studio 终端中运行：
```bash
npm install
```

### 3. 运行本地开发服务器
我们已经在 `package.json` 中移除了所有可能导致 headless 容器报错的 auto-open flags，配置了最安全的本地服务启动方案：
```bash
npm run dev
```

### 4. 预览与端口转发
* 运行后，Cloud Studio 将在控制台右下角自动弹出提示：**“检测到服务运行在端口 8080，是否打开预览？”**
* 点击 **“在浏览器中打开预览”**（或配置端口转发绑定），即可在专属的云端预览链接中体验精美流畅的交互游戏！
