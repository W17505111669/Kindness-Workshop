## 📖 项目简介

**善行工坊 · 公益宇宙** 是一款面向阿尔茨海默病适老化关怀的 HTML5 公益聚合游戏平台。项目以"奶奶的记忆阁楼"为叙事起点，融合**海洋保护、非遗传承、反诈教育、心理健康、粮食安全**六大公益主题，通过 **40 个互动游戏模块**，将公益科普与沉浸式互动深度融合，让玩家在游戏中学习公益知识、培养社会责任感。

在中国，有超过 **1000 万** 老年人正承受着阿尔茨海默病（老年痴呆症）的困扰。认知训练、生活干预、耐心的陪伴以及**回忆疗法（Reminiscence Therapy）**能有效延缓认知衰退。本项目以此为切入点，将阿尔茨海默病患者常见的"认知迷雾""记忆失焦""思绪断裂""精神孤独"等症状转化为游戏中的重构机制，以老式收音机、复古照相机、足踏缝纫机、煤油马灯等满载时代记忆的旧物为载体，带给玩家深度的共情与人文关怀。

## 🎨 视觉设计系统

### Glassmorphism 毛玻璃设计语言

全站采用统一的毛玻璃（Glassmorphism）设计风格，核心设计元素：

- **半透明面板**：`rgba(255, 255, 255, 0.03)` 背景 + `rgba(255, 255, 255, 0.08)` 边框
- **高级阴影系统**：`0 20px 40px rgba(0,0,0,0.6)` 多层次阴影叠加
- **金色辉光点缀**：`rgba(229, 169, 59, 0.4)` 琥珀金强调色
- **弹性动画曲线**：`cubic-bezier(0.68, -0.55, 0.265, 1.55)` 物理感过渡

### 字体系统

| 用途 | 字体 | 风格 |
|------|------|------|
| 标题 | Noto Serif SC | 典雅衬线，温暖人文 |
| UI / 正文 | Outfit / Inter | 现代无衬线，清晰易读 |

### 色彩方案

以**暖棕/琥珀金/玫瑰色**为基调，营造复古高级感：

- 背景：`#1c1815 → #2a2420` 深棕渐变
- 强调色：`#e5a93b` 琥珀金 / `#d48a85` 复古玫瑰
- 文字：`#f5f0eb` 暖白 / `#c2b6ab` 柔灰

## 🎵 Web Audio API 音效引擎

**零外部音频文件依赖**，全部音效基于 Web Audio API 物理合成：

### BGM 系统
- **Lo-Fi 氛围音乐**：基于爵士九和弦（Fmaj9, Cmaj9, Dm9, Am9）自动轮播
- **Rhodes 电钢琴合成**：三角波 + 正弦副振荡器 + LFO 颤音 + 低通滤波器
- **黑胶唱片噪声**：粉红噪声 + 随机脉冲模拟复古噼啪声

### 音效系统（30+ 种音效）
- 相机快门：两段式物理模拟（click-down + mirror-rebound）
- 缝纫机声：连续哒哒声循环
- 收音机效果：带通滤波模拟 AM 收音机音色
- 水晶铃声：大三度和声 + 延迟混响
- 打字机音效：随机音高模拟机械键盘
- 唤醒琶音：五声音阶扫频 C4→C5

### 文化音序器
- 🎵 **《送别》旋律生成器**：颤音 + 带通滤波，模拟老式收音机
- 🎭 **《茉莉花》戏曲风格**：93.5MHz 昆曲风格旋律
- 📻 **新闻报时信号**：102.4MHz 风格嘀嘀声

### 音频架构
```
SFX GainNode → BGM GainNode → Master GainNode → AudioDestination
```
三级音量独立控制，支持自动恢复被浏览器暂停的 AudioContext。

## 🛡️ 安全设计

### Trusted Types 安全策略

项目实现了一套生产级 HTML 消毒器（DOMPurify Lite）：

- **白名单标签过滤**：仅允许 25+ 种安全 HTML 标签
- **白名单属性过滤**：仅允许 30+ 种安全属性 + `data-*` 自定义属性
- **事件处理器剥离**：自动移除所有 `on*` 事件属性
- **危险协议阻断**：禁止 `javascript:`、`data:text/html`、`vbscript:` 协议
- **SVG 安全处理**：专用轻量正则清洗，保留 SVG 命名空间完整性

### ARIA 无障碍标准

- 全站覆盖 `aria-label`、`aria-hidden`、`aria-live`、`aria-expanded` 等无障碍标签
- Tab 导航使用 `role="tablist"` / `role="tab"` / `aria-selected` 规范
- Canvas 粒子背景标记 `role="presentation" aria-hidden="true"` 避免干扰屏幕阅读器
- 键盘快捷键支持：Esc 返回、1-5 快速切换 Tab、H/T 辅助操作

|------|------|------|
| **标记语言** | HTML5 | 语义化标签，40 个 section 屏幕 |
